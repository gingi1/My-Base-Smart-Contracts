# Interaktions-Guide: Luisli Gold (LGOLD)

Dieses Dokument erklärt, wie man mit dem Luisli Gold Smart Contract auf dem Base Mainnet interagiert.
## 🛠️ Token Details
* **Name:** Luisli Gold
* **Symbol:** LGOLD
* **Decimals:** 18
* **Network:** Base Mainnet
* **Contract Address:** `0x0cae8b7e812e46145450ea0f3048d64f091820ae`
## 🦊 Token in Wallet importieren (z.B. MetaMask)
1. Öffne deine Wallet und stelle sicher, dass du mit dem **Base Network** verbunden bist.
2. Klicke auf **"Token importieren"** (oder "Custom Token").
3. Kopiere die Adresse `0x0cae8b7e812e46145450ea0f3048d64f091820ae` hinein.
4. Die Felder LGOLD und 18 sollten sich automatisch füllen.
5. Bestätige den Import.
